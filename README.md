# EVE-NG Projects

Welcome to my **EVE-NG Projects** repository.

This repository is a collection of realistic enterprise networking projects built using **EVE-NG** to demonstrate practical networking, infrastructure, and automation skills beyond basic certification labs.

Rather than isolated exercises, each project is designed as a complete scenario that reflects real-world enterprise environments. Every project includes documentation, configurations, diagrams, screenshots, and implementation notes to showcase both the final solution and the engineering process behind it.
