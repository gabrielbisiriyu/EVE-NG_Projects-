# Project Atlas

## Overview

Project Atlas is a realistic enterprise network built in EVE-NG to demonstrate practical networking, server administration, security, and automation skills.

The project simulates the design, deployment, and operation of a growing enterprise network using Cisco IOL devices and Ubuntu Server.

## Technologies

- Cisco IOL Layer 2
- Cisco IOL Layer 3
- ASA firewall
- Ubuntu Server
- EVE-NG

## Planned Features

- VLANs
- Layer 3 Switching
- Inter-VLAN Routing
- SSH Management
- DHCP
- DNS
- File Server (Samba)
- Web Server
- NAT
- ACLs
- Site-to-Site VPN
- Firewall
- Ansible Automation
- Python Automation
- Network Monitoring
